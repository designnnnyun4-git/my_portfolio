私について

Yuna Anan

阿南 佑奈

2000年4月生まれ、大分県出身。
「自分の作ったもので、誰かを笑顔にしたい」。幼い頃から、手作りのプレゼントで家族や友人の喜ぶ顔を見るのが大好きでした。

「あの感動を仕事にしたい」と考えたとき、大好きなデザインを通じて誰かの役に立てる、Webクリエイターという道に出会いました。

ただ見た目を整えるだけでなく、お客様の目線に立ち、目的を叶えるための美しいデザインと、細部までこだわった「動き」を提供できるクリエイターを目指しています。

SKILLS

Web design

ロゴやバナー、Webサイトなど、想いを形にするデザイン制作はお任せください。

制作において一番大切にしているのは、「誰に、何を伝えたいか」という目的です。ターゲット層にしっかり刺さる魅力的なビジュアルと、ユーザーが迷わず使える設計のバランスを意識して作り込んでいます。
制作の過程ではお客様とのコミュニケーションを何より大切にし、いただいたフィードバックをしっかり反映させながら、クオリティの高い仕上がりへと丁寧にブラッシュアップしていきます。

Illustrator Photoshop Figma

Web development

お客様の想いを細部まで忠実に再現するコーディングはお任せください。

「こんな動きは無理かもしれない」「うまく言葉にできないけれど、こうしたい」といった、型にハマらないご要望にこそ、とことん向き合います。
お客様の理想を正しく理解するため密な意思疎通を図り、難易度の高い実装でも、最善の解決策を考え抜いてWebサイトへ落とし込みます。
また、見た目の再現度だけでなく、裏側の構造においても「誰が見てもわかりやすいコード」を常に心がけています。

Visual Studio Code Dreamweaver

STRENGTHS

Curiosity

探求心

昔から興味があることに対しては、納得いくまでとことん調べ尽くすタイプです。
Web制作でも「このデザインを絶対に形にしたい！」という想いを原動力に、難しい調整や複雑なコードの実装も、一つひとつ丁寧に向き合って解決することにこだわっています。

Persistence

完遂力

一度決めたゴールに対して、最後まで責任を持ってやり遂げる粘り強さがあります。
実装が難しいレイアウトや、実機特有の複雑な不具合に直面しても、決して妥協しません。原因を一つひとつ切り分け、理想の挙動が実現するまで泥臭く試行錯誤を繰り返す「完遂力」が私の武器です。

Empathy

共感力

幼い頃から「手作りのプレゼントで誰かを笑顔にする」ことが大好きでした。
この原体験を大切に、制作では常にお客様の目線に立ち、その先にいるユーザーがどう感じるかを先回りして考える「ユーザーファースト」な設計を心がけています。想いに寄り添い、課題を一緒に解決する伴走者でありたいと考えています。
